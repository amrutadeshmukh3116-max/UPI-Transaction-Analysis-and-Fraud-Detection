# UPI-Transaction-Analysis-and-Fraud-Detection
Data Cleaning and Transformation: Processed raw transaction data using excel,python(pandas) ,handling missing values and data inconsistencies and preparing structured datasets for analysis.
Exploratory Data Analysis on 100,000 UPI transaction records to identify transaction trends,fraud patterns, and system performance metrics.
Dashboard Development: Built an interactive Power BI dashboard to visualize KPIs such as transaction volume, failure trends,fraud distribution and device/merchant-level insights.
Business Insights: Generated actionable insights to support fraud monitoring,transaction reliability analysis and operational decision-making.
